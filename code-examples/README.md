# COM1001 Code Examples Folder

This folder contains Ruby/Sinatra examples discussed in lectures.

To run an example, change to its directory in the terminal and issue the
`sinatra` command. If it has a Gemfile, you may also need to do a `bundle
install` to install gems required by the application for it to work. For
instance:

```console
cd com1001-2025-26/code-examples/football_players
bundle install
sinatra
```
